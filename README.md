# Liquiwise

AI-powered financial insights and liquidity management platform designed to help users track spending, analyse financial behaviour, and make smarter money decisions through intelligent automation and data-driven insights.

---

## 🚀 Features

- 🔐 Secure authentication & user management
- 📊 Financial dashboard and analytics
- 🤖 AI-powered insights and recommendations
- 💳 Transaction and liquidity tracking
- 📈 Spending pattern analysis
- ☁️ Cloud-hosted scalable architecture
- 🐳 Dockerized backend services
- ⚡ REST API integration
- 📱 Responsive modern UI

---

## 🛠️ Tech Stack

### Frontend
- React / Next.js
- Tailwind CSS
- TypeScript

### Backend
- Node.js / Express
- REST APIs
- JWT Authentication

### Database
- PostgreSQL / MongoDB

### DevOps & Deployment
- Docker
- Render
- AWS Amplify
- GitHub Actions (CI/CD)

### AI & Analytics
- OpenAI API
- Financial data processing
- Behavioural analytics
